# 石門國小 PaGamO 閱讀素養授權填報系統 - 說明與專案總覽

**文件版本/時間**：2026.06

## 壹、 教育局公文與專案概述
* **發文單位與日期**：桃園市政府教育局，中華民國115年6月15日。
* **發文字號**：桃教資字第1150054822號。
* **專案主旨**：本市「中小學數位學習實施計畫」數位內容與教學軟體-115學年PaGamO素養學習教材需求調查暨線上說明會。
* **本系統範圍**：僅供 **桃園市石門國民小學** 校內填報使用。
* **固定學校資料**：學校全銜固定為「桃園市石門國民小學」，學校代碼固定為 `034725`。
* **申請老師資料**：每筆班級需求需填寫申請老師姓名，供後台追蹤需求來源。
* **教材開放科目**：「中文閱讀素養」、「英文閱讀素養」及「數學素養」3項科目。
* **申請彈性**：115學年度教材調整為「各科目可單獨申請」模式，校內教師得依班級需求，自由混搭申請1至2項學科教材。
* **填報期限**：請石門國小校內教師或業務承辦人於 **115年7月3日(星期五)** 前完成填報。
* **校內聯絡**：校內填報問題請聯繫資訊組長。

## 貳、 桃園市素養專案總覽與使用概況
* **登入方式**：教育雲 (OPEN ID) 縣市帳號登入世界「桃園市_PaGamO素養學習」。
* **派發時間**：學期間每週派發一篇（不含寒暑假）。
    * 中文素養：週一
    * 英文素養：週二
    * 數學素養：週三
* **石門國小填報年級**：本系統依教材建議只開放國小 3~6 年級填報。
    * 中文素養：國小建議 3~6年級適用，每週1篇。
    * 英文素養：國小建議 4~6年級適用，包含閱讀素養與文法。
    * 數學素養：國小建議 5~6年級適用。
* **平台使用數據 (至113學年統計)**：
    * **授權與啟用**：18,225位學生獲得教材授權，16,667個帳號已啟用（獲授權學生90%有登入使用），比去年成長 8,250人。中文教材啟用率達 81%。
    * **活躍度**：10,418位學生每週活躍使用，每3人有1名學生穩定使用。
    * **累積學習**：累積使用 101,313 小時，建立 4,090 個班級（覆蓋全市53%的班級），作答 23 萬篇素養任務（以中文閱讀素養為主，約13萬篇）。
    * **平均投入**：這學期平均每個學生有 3.3 小時在 PaGamO 上學習。

## 參、 各科教材介紹與特色

### 1. 中文閱讀素養
* **學習目標**：精準訓練圖表文、高層次閱讀理解能力。
* **文本類型**：跨領域與連結生活情境文本（四大領域：自然科學、社會科學、人文史地、生活知識），包含連續文本與非連續文本（如圖表、電影時刻表等）。
* **提問設計 (探究性理解歷程)**：
    * 擷取訊息 (WHAT 事實性提問)：文本表層可見的事實資訊。
    * 統整解釋 (HOW 推論性提問)：文本可見的結構脈絡與推論的個別意義。
    * 省思評鑑 (WHY、HOW 評論性提問)：核心概念與反思應用。
* **平台功能**：支援注音模式、可劃記重點、可調整文字大小。並提供個人能力表現的雷達圖數據追蹤。

### 2. 英文閱讀 / 英語文法課
* **英語文法課**：石門國小以國小中、高年級需求為主。每單元皆包含真人文法教學影片，透過聽力與閱讀交叉訓練確實掌握文法使用（如 be 動詞現在式）。
* **英語閱讀素養**：石門國小以國小中、高年級需求為主。精選跨領域文章，符合108課綱趨勢，提升閱讀理解速度。
* **校園導入案例 (將 PaGamO 成為獎勵)**：
    * 課前共同約定：若達成教師指定課堂任務，下一堂可使用平板練習。
    * 課中獎勵：全班上課認真，下課前給予10-20分鐘時間玩中學。
    * 數據掌握：課後至後台查看學習成效，有效掌握學生錯題並安排複習。

### 3. 數學素養
* **答題特色**：題組提供多元題型包含選擇題和「填充題」，有效檢核學生是否真正學會，避免單一題型造成瞎猜。答題介面提供專屬「數學符號鍵盤」幫助學生輕鬆輸入答案。
* **計算紙功能**：提供便捷計算方式，學生可自由移動與縮放邊計算邊對照題目。系統會完整紀錄計算紙內容，方便教師查看學生計算歷程。
* **訂正與補救**：
    * 提供官方詳解與玩家圖文詳解，引導學生重新思考解題脈絡，訂正錯題。
    * 延伸挑戰：完成素養任務後，根據任務表現推薦對應的自選章節，引導學生前往挑戰把觀念練習到精熟。

## 肆、 115-1 學年素養教材申請與填表說明
* **申請方式**：石門國小需於 7/3(五) 前彙整並回傳「班級授權清冊 (Excel表格)」至「Google表單 (線上問卷)」。
* **填寫規範**：
    * 石門國小僅需回傳一份班級授權清冊，請以「班級」為單位填入。
    * 本次調查每班最多申請 2 科，教育局審核後若有剩餘授權名額，將開放第二次申請 (可申請第3科)。
* **常見錯誤提醒**：
    * 錯誤一：學校代碼錯誤（石門國小學校代碼固定為 `034725`）。
    * 錯誤二：班級重複（同一分頁內有兩個相同年級與班級的資料）。
* **科任老師跨多班填表說明**： Excel 清冊中必須「一班填寫一列」。例如英文科任老師同時任教 5 個班級，就必須分成 5 列獨立填寫，以利後台精準匯入各班學籍。

## 伍、 常見問答 (Q & A)
* **Q1：校管權限如何申請？沒看到任教班級作答狀況怎麼辦？**
    * A1：系統連動自動開通，只要學校校務系統中已有正確設定任教學科與班級，系統會自動開通對應權限。若確認設定妥當仍無權限，請另外填寫《班級管理權限申請表》。
* **Q2：請問這次上下學期都要填寫嗎？**
    * A2：目前只需填寫 115-1 學期。下學期如有異動，會在開學前再度調查填寫。
* **Q3：表單提交後如果又有老師想加簽申請，還能修改嗎？**
    * A3：可以，一律以最新上傳版本為準。截止日 (7/3) 前若有異動，承辦人只需將修改好的 Excel 重新提交 Google 表單，系統將以最後一次送出的最新紀錄為準。

   - 若資料量或使用範圍擴大，可改成 GAS / Cloud Functions / Supabase Edge Function 代理。

3. **RLS 權限強化**
   - 正式部署可限制前台只能新增/更新，不開放公開查詢。
   - 後台使用額外服務端權限讀取資料。

4. **資料備份機制**
   - 定期把 Supabase 填報資料匯出到 Google Sheets 或下載備份。
   - 避免截止日前資料誤刪或服務異常。
